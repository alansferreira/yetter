### 1.0.1 (2026-07-21)

## [1.0.0] - 2026-07-21

- Initial public release.
- YAML multi-document updates with `$doc[n]` selector support.
- Document and node-level filter support with `[?(...)]` using `==` and `!=`.
- Typed value coercion for `string`, `number`, `boolean`, `null`, `array`, `object`.
